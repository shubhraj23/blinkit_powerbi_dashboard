# 📊 Blinkit Sales Dashboard (Power BI)

## 🔍 Overview

This project analyzes Blinkit's sales performance using Power BI. The dashboard provides a comprehensive view of key business metrics such as total sales, average sales, number of items, and customer ratings.

## 📌 Key Metrics

* **Total Sales:** $1.20M
* **Average Sales:** $141
* **Number of Items:** 8523
* **Average Rating:** 3.9

## ❓ Business Questions Solved

* What is the overall sales performance of Blinkit?
* Which product categories contribute the most to revenue?
* How does customer rating impact sales performance?
* What is the average sales per item?
* How many items are sold across different categories?

## 📊 Dashboard Features

* Interactive slicers for dynamic filtering
* KPI cards for quick performance tracking
* Clean and modern UI design
* User-friendly layout for better insights
* Scalable design for multi-page dashboards

## 💡 Key Insights

* High-rated products tend to generate better sales performance
* A few categories contribute significantly to total revenue
* Many items have lower individual sales contribution
* Average sales reflect demand and pricing balance
